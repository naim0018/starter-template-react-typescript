## How to Use

### For AI Agents

1. Read `AGENTS.md` in the project root for complete project context
2. Reference specific skill files for detailed patterns and examples
3. Follow the rules and conventions defined in each skill

### For Developers

1. Use these files as reference when implementing features
2. Follow the patterns and checklists in each skill
3. Keep skills updated as the project evolves
